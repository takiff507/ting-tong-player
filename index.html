Installation & Setup Guide

×
1

Install the APK File

Open the downloaded ting_tong_player_safe.apk file. If prompted by Android, allow installation from "Unknown Sources" in your device's security settings.

2

Browse Default Streams

Open the app. You will instantly find 100% legal, preloaded Free-to-Air (FTA) and FAST streams (like DD Sports, Red Bull TV, WWE) ready to play in the player.

3

Import Custom M3U Playlists

To watch custom streams, tap the Settings (Gear Icon) in the top-right corner. Select Import M3U Playlist, paste your custom playlist link (.m3u or .m3u8), and tap Save. The app will fetch and organize your streams locally!

4

Set Up Proxy Tunnels (Optional)

If you want to protect your personal IP address while playing streams, configure your HTTP proxy address in Settings under the Proxy Settings section.
