# Ting Tong Player

Static landing page for Ting Tong Player, a private media player client for personal stream links.

Live site after GitHub Pages deployment:

https://takiff507.github.io/ting-tong-player/

Deployment is handled by the GitHub Pages workflow in `.github/workflows/pages.yml`.

APK downloads are linked through the repository Releases page so release assets can be managed separately from the website.
